
# Тест API 


| Тип запроса                                                            | Тестируемый модуль         | Цель теста                               | Негативный / позитивный | Ожидаемый результат                                                                     | Актуальный результат | Результат |
| ---------------------------------------------------------------------- | -------------------------- | ---------------------------------------- | ----------------------- | --------------------------------------------------------------------------------------- | -------------------- | --------- |
| POST /api/competitions {valid Competition}                             | POST competitions          | Создание соревнования                    | Позитивный              | 201 Created + тело с сохранённым Competition, файл JSON обновлён                        |                      |           |
| POST /api/competitions {duplicate Id}                                  | POST competitions          | Дубликат соревнования ломает репозиторий | Негативный (ломающий)   | Во время выполнения — ArgumentException, тест ожидает падение (Assert.Throws)           |                      |           |
| POST /api/competitions {invalid JSON}                                  | POST competitions          | Некорректный JSON                        | Негативный              | 400 BadRequest от пайплайна модели                                                      |                      |           |
| GET /api/competitions                                                  | GET competitions           | Получение всех соревнований              | Позитивный              | 200 OK + список (>= 1 элемента после POST)                                              |                      |           |
| GET /api/competitions/{existing-guid}                                  | GET competition by id      | Получение существующего соревнования     | Позитивный              | 200 OK + правильный объект                                                              |                      |           |
| GET /api/competitions/{non-existing-guid}                              | GET competition by id      | Не существующее соревнование             | Негативный              | 404 NotFound                                                                            |                      |           |
| PUT /api/competitions/{id} c тем же Id в теле                          | PUT competitions           | Успешное обновление                      | Позитивный              | 204 NoContent, повторный GET возвращает обновлённые поля                                |                      |           |
| PUT /api/competitions/{id} c другим Id в теле                          | PUT competitions           | Несовпадающий Id                         | Негативный              | 400 BadRequest                                                                          |                      |           |
| PUT /api/competitions/{id} для несуществующего соревнования            | PUT competitions           | Обновление несуществующего               | Негативный (ломающий)   | ArgumentException в репозитории, тест ждёт исключение / 500                             |                      |           |
| DELETE /api/competitions/{existing-guid}                               | DELETE competitions        | Удаление существующего                   | Позитивный              | 204 NoContent, последующий GET /api/competitions/{id} -> 404                            |                      |           |
| DELETE /api/competitions/{non-existing-guid}                           | DELETE competitions        | Удаление несуществующего                 | Негативный              | 204 NoContent, список не меняется                                                       |                      |           |
| POST /api/results {valid CompetitionResult}                            | POST results               | Создание результата                      | Позитивный              | 201 Created + тело с Result                                                             |                      |           |
| POST /api/results {duplicate Id}                                       | POST results               | Дубликат результата ломает репозиторий   | Негативный (ломающий)   | ArgumentException из репозитория, тест ждёт падение                                     |                      |           |
| GET /api/results                                                       | GET results                | Получение всех результатов               | Позитивный              | 200 OK + список (>= 1 после POST)                                                       |                      |           |
| GET /api/results/{existing-guid}                                       | GET result by id           | Получение существующего результата       | Позитивный              | 200 OK + корректный Result                                                              |                      |           |
| GET /api/results/{non-existing-guid}                                   | GET result by id           | Не существующий результат                | Негативный              | 404 NotFound                                                                            |                      |           |
| GET /api/competitions/{existing-competitionId}/results                 | GET results by competition | Результаты по конкретному соревнованию   | Позитивный              | 200 OK + список только с этим CompetitionId                                             |                      |           |
| GET /api/competitions/{non-existing-competitionId}/results             | GET results by competition | Пустой результат                         | Позитивный              | 200 OK + пустой массив                                                                  |                      |           |
| PUT /api/result/{id} с тем же Id                                       | PUT results                | Успешное обновление результата           | Позитивный              | 204 NoContent, GET возвращает обновлённый Result                                        |                      |           |
| PUT /api/result/{id} с другим Id                                       | PUT results                | Несовпадающий Id                         | Негативный              | 400 BadRequest                                                                          |                      |           |
| PUT /api/result/{id} по несуществующему Result                         | PUT results                | Обновление несуществующего               | Негативный (ломающий)   | ArgumentException в репозитории, тест ждёт исключение / 500                             |                      |           |
| DELETE /api/results/{existing-guid}                                    | DELETE results             | Удаление результата                      | Позитивный              | 204 NoContent, повторный GET /api/results/{id} -> 404                                   |                      |           |
| DELETE /api/results/{non-existing-guid}                                | DELETE results             | Удаление несуществующего                 | Негативный              | 204 NoContent, список не меняется                                                       |                      |           |
| GET /api/competitions/{id}/results при баге (нет return внутри лямбды) | GET results by competition | Ломающее поведение эндпоинта             | Негативный (ломающий)   | Тест ожидает 500 или падение из‑за null‑результата; фикс контролируется изменением кода |                      |           |
